\noindent \textbf{1}
In metric topology, metric is a continuous function. Also, the absolute value have image in $\mathbb{R}$ by $\{p^n:n\in\mathbb{Z}\}$. Therefore, for any $r>0$, there exists $N\in\mathbb{Z}$ such that $p^{N-1}< r \leq p^N$ and $B(0,r) = \{b\in\mathbb{Q}_p:\abs{b}\leq p^{N-1}\}=\{b\in\mathbb{Q}_p:\abs{b}< p^{N}\}$. It means that $B(0,r)$ is a closed and open set for $r>0$. By checking the definition of $B(a,r)$, we get $B(a,r)=a+B(0,r)$, and as multiplying by constant homeomorphism, continuous function with continuous inverse by adding $-a$, $B(a,r)$ is an open and closed set.\\

\noindent \textbf{2}
Choose $b\in B(a,r)$. If $r=0$, then $b=a$, so I'll assume $r>0$. For $y\in B(a,r)$, $\abs{b-y} \leq \max(\abs{b-a}, \abs{a-y})<r$, so $y\in B(b,r)$. Conversely, if $y\in B(b,r)$, then $\abs{a-y}\leq \max(\abs{a-b}, \abs{b-y})<r$, so $y\in B(a,r)$. Therefore, $B(a,r)=B(b,r)$.
